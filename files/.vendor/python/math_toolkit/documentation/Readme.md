Static notebook documentation shipped with `math_toolkit` lives here.

Directory structure:

- `index.ipynb` is the reader-facing documentation map.
- `library/` contains concise reference pages reached from `Help(obj)`.
- `tutorials/` contains general tutorials for practical toolkit tasks.
- `worksheets/` contains longer mathematical investigations.

Authoring guidance:

- `AUTHORING_GUIDE.md` gives the shared principles and points to page-specific
  guidance.
- `LIBRARY_GUIDE.md` covers concise topic pages.
- `TUTORIAL_GUIDE.md` covers general tutorials and mathematical worksheets.
